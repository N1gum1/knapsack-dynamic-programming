# knapsack-dynamic-programming
Implementação do Problema da Mochila (Knapsack Problem) em Python utilizando programação dinâmica. O projeto calcula o valor máximo que pode ser obtido respeitando o limite de peso, identifica os itens escolhidos e apresenta visualização dos resultados por meio de tabela e gráfico.

Este projeto implementa uma solução para o Problema da Mochila utilizando programação dinâmica em Python, com visualização dos resultados.

Descrição
Dado um conjunto de itens, cada um com um peso e um valor, e uma mochila com capacidade limitada, o objetivo é determinar o maior valor possível sem ultrapassar o limite de peso.
Funcionalidades
Cálculo do valor máximo
Identificação dos itens selecionados
Exibição da tabela de programação dinâmica
Visualização gráfica dos itens escolhidos

Tecnologias
Python 3
Matplotlib
